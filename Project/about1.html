About the game

This game was made as a part of my CS25210 assignment. The game was created using CSS, HTML5 canvas and JavaScript. The game is a spaceship that avoids the meteors and uses laser cannons to destroy them. All images for the game are created using the canvas. The game features levels that get progressingly harder

How to play

The objective of the game is to monouver the spaceship around the meteors while firing at them and destroying them with the lasers. The spaceship is controlled using the W, A, S and D keys on the keyboard and the lasers are aimed using the cursor and fired using the left click. The aim of the game is to complete all four levels which get progressingly more difficult.
